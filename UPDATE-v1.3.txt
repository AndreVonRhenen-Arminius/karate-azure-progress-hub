HOW TO UPDATE GITHUB

1. Extract Karate-Azure-Progress-Hub-v1.3.zip.
2. Open your GitHub repository: karate-azure-progress-hub.
3. Select Add file > Upload files.
4. Upload all files and folders from inside the extracted karate-azure-progress-hub folder.
5. Allow GitHub to replace files with the same names.
6. Commit the changes with the message: Update to morning programme v1.3
7. Wait for the GitHub Pages deployment to show a green tick.

INSTALLED APP UPDATE

You do not need to uninstall or install the PWA again.

Phone:
- Fully close the app.
- Open it again while connected to the internet.
- If the old plan still appears, open the website in Chrome and refresh once, then reopen the installed app.

Windows PC:
- Close the installed app completely.
- Open it again while connected to the internet.
- If the old plan still appears, press Ctrl+R once or open the website in Edge and refresh.

The installed phone and PC apps use the same GitHub Pages website, so replacing the GitHub files updates every installed copy.
