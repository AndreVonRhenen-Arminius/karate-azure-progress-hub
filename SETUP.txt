KARATE & AZURE PROGRESS HUB v1.7.1

LOCAL USE
1. Double-click run-local.bat.
2. Open http://localhost:8080/.
3. Local progress is saved automatically in the browser.

MICROSOFT ONEDRIVE
1. Follow MICROSOFT-ONEDRIVE-SETUP.md.
2. Register http://localhost:8080/redirect.html and the deployed HTTPS redirect.html URL as SPA redirect URIs.
3. Add delegated permission Files.ReadWrite.AppFolder.
4. Enter the Entra Application client ID in Settings.
5. Select Microsoft OneDrive and sign in.

GITHUB PAGES
Upload the complete folder to the repository root. Do not overwrite an existing js/config.js file unless explicitly intended.

SUPABASE
The previous Supabase configuration, authentication and user_app_state table remain supported and unchanged.

VALIDATION
- node tests/smoke-test.cjs
- node tests/onedrive-sync-test.cjs

SECURITY
Never add a Microsoft client secret, Supabase service-role key, access token or refresh token to browser files.
