TO UPDATE THE LIVE APP
1. Extract the ZIP.
2. Open the existing GitHub repository.
3. Select Add file > Upload files.
4. Upload everything from inside the extracted app folder.
5. Commit the replacements.
6. Wait for GitHub Pages deployment to complete.
7. Close and reopen the installed app on each device.

No reinstallation is normally required.

LOCAL TEST
Double-click run-local.bat, then open http://localhost:8080.
